```js
document.getElementById("num").addEventListener("blur", checkEmptyAndFill);
```
`checkEmptyAndFill`是一个函数引用，函数本身没有被执行。当发生`blur`事件，浏览器会调用`checkEmptyAndFill`函数。
```js
document.getElementById("num").addEventListener("blur", checkEmptyAndFill());
```
`checkEmptyAndFill()`是在这行代码执行时立即运行，不是在事件发生时。函数立即被调用，执行后的返回值会被传递给`addEventListener`
