# 穷爸爸 富爸爸

穷爸爸：
我永远也不会成为富人。
富爸爸：
把自己说成一个富人，他会说：我是一个富人，富人不会这么做。
一次严重的经济挫折让他一问不名，他会安慰自己：贫穷和破产的区别是，破产是暂时的，而贫穷是永久的。

穷爸爸之所以穷，不在于挣钱的多少（这也很重要），而在于他的想法和行为。
一个爸爸说——努力学习，获得学位，找个工资高的工作，成为一名专业人士，律师、会计师、读MBA；
另一个爸爸说——学习成为富人，了解钱的运动规律并让钱为我工作。“我不为钱工作，我要让钱为我工作。”

# 克服困难

## 恐惧心理

害怕亏钱 or 讨厌冒险 尽早行动并且制定一个完善的退休计划 聘请一位信得过的财务规划师 做出任何投资决定之前，需要指导

德州，什么东西都大气。德克萨斯人赢了，赢很多；输了，也输的令人吃惊。
德克萨斯人对于风险、收益和失败的态度，这是他们驾驭生活的方式，活得很大气，而非大部分人碰见金钱问题时，在杂货店店员少找两毛五分钱就抱怨不停。
德克萨斯人的生活态度，赢了会骄傲，输了也不忘吹牛。有句谚语，如果你要破产了，那就索性一分不剩。他们不愿让我们以为他们仅仅因为一幢复式公寓而破产。

大部分人在财务上不成功的最大原因是他们的做法过于安全，“太害怕失败所以才会失败。”
> 胜利意味着不害怕失败。
> 
 >- 美国职业橄榄球联赛四分卫 弗朗·塔肯顿

想学习如何面对风险、损失和失败，就去圣安东尼奥市走访阿拉莫。
阿拉莫故事讲的是，一群勇敢的人在知道毫无胜算的情况下依然选择战斗，宁可战死也不愿投降。
这是一个值得学习的、振奋人心的故事，这也是一次悲壮的军事失败，他们不得不用枪托进行肉搏，他们最终失守了阿拉莫。德克萨斯人面对失败时高声呼喊：记住阿拉莫！

这个故事给了富爸爸力量，因为它总是在提醒他，他可以将财务上的损失变成赢利。富爸爸知道失败会使他变强大，变精明。他不愿意亏钱，但清楚自己是什么样的人，知道怎么面对损失。接受损失，然后将它变成赢利，这是他成为赢家而别人沦为失败者的原因所在，同时这也是当别人退出时，依然有勇气冲过终点线的原因。
“德克萨斯人接受失败的现实并把它转变成通往成功道路上的一个个插曲。”更深一点的体会是，“德克萨斯人并不掩饰失败，愈挫愈奋，他们接受失败的现实并将失败转化为动力。失败激励德克萨斯人走向成功。”

对于胜利者，失败会激励他们；对于失败者，失败则会击垮他们。

失败会激励胜利者，击败失败者。这是胜利者最大的秘密，也是失败者所不知道的秘密。“胜利意味着不害怕失败。”
不害怕失败，因为他们了解自己。他们和所有人一样不喜欢失败，但他们知道，失败会激励他们做的更好。不喜欢失败和害怕失败之间有着巨大的差别，大部分人因为害怕失败而失败。他们在财务上做的过于安全、规模太小，买大房子、大轿车而不进行大的投资。90%以上美国人财务困难的主要原因，为了避免损失而理财，而不是为了赢利而理财。

选择购买一个安全的投资组合，将大量现金以存款单、低收益债券、可以在共同基金内部买卖的共同基金和一点股票的形式进行投资。但这却不是一个赢利的投资组合。这种投资组合只是人们为避免损失而采取的方式。

安全的投资组合比没有强很多，这对于注重安全的人是很合适的，但是，安全的、平衡的投资于一个投资组合，却不是一个成功的投资者应有的投资方式。资金很少而又想致富，必须首先集中于一点，而不是追求平衡。要进步，必须先做到不平衡，并注意怎样才能使自己不断进步。如果有致富的欲望，必须集中精力，把大部分鸡蛋放在较少的篮子里，而不是像穷人和中产阶级一样，把很少的鸡蛋放在许多篮子里。

如果不愿失败，就安全的投资；如果损失会使我元气大伤，稳妥一点，做平衡的投资，要是超过了25岁害怕冒险，那就不要改变自己的投资方式。安全的方式投资，但要尽早起步，早点开始积累你的鸡蛋，以这种方式积累需要大量的时间。

如果想要获得自由，从老鼠赛跑的游戏解脱出来，应该问自己的第一个问题：我该如何去面对失败？如果失败能争取胜利，也许去争取每一次投资机会（仅仅是也许而已）；如果失败会损失惨重or烦躁不安，像个愣头青一样，只要不如意就找律师帮忙诉讼，还是稳妥一点，继续工作，购买一些国债和共同基金，但要记住，这些金融工具同样有风险，虽然较为安全。

积累资产项很容易，这就像一场低智商的游戏，五年级的数学水平就够了。然而，用资产进行投资却是一种高智商的游戏，需要胆量、耐心和对待失败的良好态度。失败者回避失败，而失败本来是可以使失败者转变为成功者，一定要记住“阿拉莫”。
